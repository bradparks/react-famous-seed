# react-famous-seed


Seed project for [react-famous](https://github.com/pilwon/react-famous) apps.


## Requirement

    npm install -g gulp


## Usage

    git clone https://github.com/pilwon/react-famous-seed.git
    cd react-famous-seed
    npm install

### Development

    gulp dev
    # http://localhost:8080

### Production

    gulp build
    # open dist/index.html
